
### **AWS Compute – Quick Memo Table**

| Service                               | One-line memo                                |
| ------------------------------------- | -------------------------------------------- |
| **EC2**                               | Virtual servers with full OS control         |
| **Lightsail**                         | Simplified EC2 for small apps & VPS          |
| **Lambda**                            | Serverless functions, event-driven           |
| **AWS Batch**                         | Run large batch & parallel jobs              |
| **Elastic Beanstalk**                 | PaaS for web apps (EC2 underneath)           |
| **Serverless Application Repository** | Prebuilt serverless app templates            |
| **AWS Outposts**                      | AWS infrastructure in your datacenter        |
| **EC2 Image Builder**                 | Automate AMI creation & patching             |
| **AWS App Runner**                    | Run web apps/APIs from code or container     |
| **AWS SimSpace Weaver**               | Large-scale spatial simulations              |
| **Parallel Computing Service (PCS)**  | Managed HPC / parallel computing             |
| **AWS Global View**                   | Central visibility across regions & accounts |

---

### **Ultra-fast grouping (memory trick)**

* **VMs** → EC2, Lightsail
* **Serverless** → Lambda, App Runner
* **Batch / HPC** → Batch, PCS
* **Platforms** → Elastic Beanstalk
* **Specialized** → Outposts, SimSpace Weaver
* **Support / Ops** → Image Builder
* **Discovery / Visibility** → Global View
* **Templates** → Serverless App Repo
